Команди для Git 
git clone (открыть репозиторий в студии)
git checkout -b develop
git status (проверяет на изменения в проект) 
git push --set-upstream origin develop (перейти в репозитории в другую ветку) 
git add .  ( додати всі файли і папки в комміт)
git commit -m (зберегти зміни)
git push (відправити зміни команді)

Команди для папок
cd (перейти в папку) .. (в гору)
code . (відкрити папку в VS)


Команди для yarn "загрущік бібліотек"
yarn add (завантажити бібліотеку)
yarn remove (видалити бібліотеку)
yarn install (оновити бібліотеки)

Команди для parcel "збірник файлів"
parcel (шлях до файлу "html")
http://localhost:1234 (адреса сервера)
